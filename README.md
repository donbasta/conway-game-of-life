# Conway Game of Life

### Description
Simple Game of Life invented by John Conway.
The rule of the game can be read 
[here](https://en.wikipedia.org/wiki/Conway%27s_Game_of_Life).

### Requirement
- Gradle 7.2
- Java (JDK 11)
- JUnit 5

### How to test
```bash
./gradlew test
```

### How to build
```bash
./gradlew build
```

### How to run the app
```bash
./gradlew run
```
